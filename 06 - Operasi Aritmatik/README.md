# Operasi Aritmatik

| Operator      | Nama |Contoh|
| ----------- | ----------- |-----|
|`+`|Tambah|`x + y`|
|`-`|Tolak|`x - y`|
|`*`|Darab|`x * y`|
|`/`|Bahagi|`x / y`|
|`%`|Modulus(baki daripada bahagi)|`x % y`|
|`**`|Eksponen(kuasa)|`x ** y`|
|`//`| Floor division(hasil bahagi dibundarkan ke nilai terbawah)|`x // y`|

## Turutan Pengiraan
Pengiraan di python yan diutamakan:
1. `()`
2. `**`
3. `*`, `/`, `%`,`//`
4. `+`,`-`

## Output Main.py :
```
10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3.3333333333333335
10 ** 3 = 1000
10 % 3 = 1
```
